## Hướng dẫn chạy project

**Bước 1**: Cài đặt [Node.js](https://nodejs.org/en/)

**Bước 2**: Clone project về máy
```sh
git clone https://github.com/levanvux/eShelf.git
```
**Bước 3**: Chạy các lệnh sau trong terminal
```sh
npm install
npm run dev
```
